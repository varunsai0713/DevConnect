# DevConnect

DevConnect is a professional collaboration platform tailored specifically for software developers and engineers. It provides a modern social environment to share project ideas, showcase work, connect with like-minded peers, and discuss the latest industry trends.

---

## 🚀 Key Features

*   **Sleek Onboarding & Auth Flows**: Clean and responsive landing page, secure Sign-In, and Sign-Up user experiences.
*   **Account Protection**: Comprehensive email verification and password recovery systems to ensure safe access.
*   **Rich Profile Configuration**: Tailor-made developer details section where users can outline their biography, professional goals, tech stacks, and social links.
*   **Intuitive Collaboration Dashboard**: A hub for exploring other developers' profiles, discovering new projects, and starting meaningful connections.
*   **Fluid Motion Transitions**: Powered by modern animation states for natural and professional visual feedback.

---

## 🛠️ Built With

*   **Frontend Library**: [React 19](https://react.dev/) / Vite 6 for lightning-fast build cycles.
*   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) for fully fluid, mobile-responsive layouts.
*   **Animations**: [Motion](https://motion.dev/) for elegant component and route transitions.
*   **Iconography**: [Lucide React](https://lucide.dev/) for crisp, consistent vector visuals.
*   **State Management**: React Context API for lightweight, component-level global states.
